But du projet

Parcourir le web

Le but du premier travail est d'écrire un programme qui parcourt des pages web en traversant des hyperliens tout en :

  • Sauvegardant localement les pages explorées
  • Collectant les adresses courriels trouvées dans les pages

Le travail vise à démontrer vos capacités de programmation en Java.

Phases du projet
Phase 1

2 points Structure du projet

Structure de votre projet
Vous devez créer un répertoire avec votre nom de famille qui devra lui-même contenir 2 sous répertoires. Un pour les fichiers CSS que vous produirez et un autre pour les images qui illustreront votre site. Vous devez ensuite créer le fichier texte index.html pour votre page d'accueil. La structure finale devra ressembler à (pour un étudiant appelé Dupont)
  • Dupont/
    • index.html
    • css/
    • images/
Pointage
  1. Aucune structure : un seul fichier ou aucun respect des noms
  2. Structure partiellement respectée
  3. Structure identique à la demande/

3 points Structure de la page index.html : head

Structure de la balise head
Vous devez utliser l'encodage UTF-8 pour l'ensemble de votre page.
Vous devez donner le titre suivante à votre page "Professions informatiques : accueil"
Pointage
  • 1 point : structure
  • 1 point : titre
  • 1 point : déclaration encodage

6 points Structure de la page index.html : body

Structure de la balise body
Vous devez reproduire la section suivante:
  • body
    • div avec id top
    • div avec id middle
      • div avec id content
        • un titre de niveau h1 : "Moi prénom nom"
        • un paragraphe vide
        • un titre de niveau h1 : "Mon projet"
        • un paragraphe vide
      • div avec id menu
    • div avec id bottom
Pointage
  • 1 point pour le top
  • 4 points pour le middle
    • 3 points pour le content
    • 1 point pour le menu
  • 1 point pour le bottom

2 points Contenu de la section Moi

Vous devez vous présenter en imaginant votre situation dans 10 ans. Vous devez produire un texte contenant entre 80 et 120 mots.

2 points Contenu de la section "Mon projet"

Vous devez décrire le projet en mentionnant:
  • l'intitulé du cours ainsi que son numéro
  • l'objectif d'apprentissage des différents métiers et formations en informatique
  • L'objectif d'apprentissage de HTML et CSS
Votre texte doit faire entre 80 et 120 mots.
