Objectif

Ce travail consiste à programmer un service pour gérer un tiroir caisse et le calcul du change nécessaire au paiement d'un achat.

Vous devrez programmer les éléments demandés mais également mettre en place des tests automatiques afin de démontrer la qualité du code produit.

Couverture de code

5 points test Regroupez tout le code que vous avez produit dans un unique package pour faciliter le calcul de la couverture de code.

  • 0 point en dessous de 50%
  • 1 point entre 50% et 60%
  • 2 points entre 60% et 70%
  • 3 points entre 70 et 80%
  • 4 points entre 80 et 95%
  • 5 points > 95%
