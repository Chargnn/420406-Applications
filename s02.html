Semaine 2 : Programmation impérative (procédurale) en Java

Avant la séance (2h)
Vous devez parcourir la page Wikipedia concernant HTML (max 20 minutes).
Vous devez parcourir le site officiel (en anglais) fondamentaux au Java (max 20 minutes).
Vous devez regarder la video suivante:
Ensuite vous devez commencer les exercices à droite. Arrêtez vous quand vous avez complété vos 2 heures.
exercice 1
Le premier exercice consiste à reproduire la page qui est produite dans la video. Faite un fichier appelé simpsons.html
exercice 2
Vous devez créer une page que vous sauverez dans un fichier premierepage.html. Vous devez mettre votre nom comme titre de la page. Son contenu doit contenir au moins 3 titres de niveaux différents (h1 à h6) ainsi qu'une liste HTML numérotée et une liste html non numérotée.
exercice 3
Utilisez le validateur HTML5 pour corriger les pages des exercices précédents.
Pendant la séance Introduction à HTML et à l'environnement de développement. Première page, première balises. Les exercices à faire seront les suivants.
