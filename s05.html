Exercices de la semaine :

Pour chaque exercice, vous devrez programmer une classe de code et une classe de tests. Utilisez la couverture de code pour valider que le code est testé en entier.

Exercice Fibonacci

Le programmeur doit programmer la suite de Fibonacci en 2 versions. 

BigInteger fibo(BigInteger biggie);
Long fibo(Long longie);

Le testeur doit écrire des tests pour s'assurer du bon fonctionnement. Quelques indications:

  • le résultat doit toujours être positif
  • le fibo d'un nombre est toujours plus grand que le fibo du nombre précédent
  • si je fais fibo(n)-fibo(n-1) je devrais trouver la valeur de fibo(n-2)

Exercice CodePostal

Le programmeur doit programmer la méthode suivante qui indique si un code postal canadien est valide. 

boolean isValidZipCode(String zippy);

Le testeur doit valider cette méthode.

  • on accepte les minuscules et majuscules
  • les 2 blocs peuvent être séparés par 0 1 ou 2 espaces ou un tiret
  • Ok "H1Y1E1" "H1Y 1E1" "h3Y 1E8"
  • Ko "H1Y 1E1" "H1Y_1E1" "h3Y 1TO"
