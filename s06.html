Écrivez les tests pour valider le bon fonctionnement du code que vous aviez écrit. Apportez les correctifs nécessaires. Validez le bon fonctionnement de votre application console existante.

Écrivez les tests qui simulent une classe de 20 étudiants avec 3 épreuves avec des épreuves au hasard. Tirez des pointages d'épreuve entre 20 et 40. En tirant des notes aléatoires, assurez vous que la moyenne est aux alentours de 50%. Assurez vous de rendre vos tests reproductibles.
