Exercice TapeLeLapin

Vous devez programmer le jeu de Tape Le Lapin. Votre application doit être i18n compatible avec des versions anglais et français.

  • Au début du jeu, le lapin est sur une case
  • Le joueur tape sur un bouton 
    • S’il tape le lapin, on augmente le nombre de pafs et le lapin change de case
    • S’il rate, on compte un flop mais le lapin reste où il est
