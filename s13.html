Exercices de la semaine

Exercice EvenementsMultiple

Programmez une application avec un bouton qui lance un évémements via un post sur un bus. Programmez un Subscribe dans l'activité de l'application qui affiche un Log.

Après avoir passé plusieurs fois la tablette du mode paysage au mode portrait puis au mode paysage. Appuyez sur le bouton et observez le Log. Y a-t-il plusieurs Log pour chaque événements? Expliquez.

Exercice EvenementsHéritage

Programmez une application avec 3 boutons qui lance respectivement un événement de type:

  • classe A qui contient un champ de type date correspondant à la date d'appui sur le bouton
  • classe A1 une sous-classe de A
  • classe A2 une autre sous-classe de A

Écrivez 3 subscribe affichant des messages différents selon le type d'événements dans le Log. En déclenchant les différents événements (en appuyant sur les boutons), observez le Log et expliquez.
